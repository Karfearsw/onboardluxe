CRM Walkthrough

New Hire · SDR · Agent Prerequisites: Lessons 02 & 03 Assessment: Hands-on Checklist

Why the CRM Matters

The CRM is how Ocean Luxe stays coordinated across the team. Clean data and consistent logging protect the customer experience and prevent leads from falling through the cracks. Every missed log is a potential missed deal — or a bad handoff to a teammate.

The Three Non-Negotiables
1. Required fields completed so the record is usable
2. Every outreach attempt logged so activity is visible
3. Every next step scheduled so nothing sits "in limbo"

A) Create or Update a Lead

  1. Search first. Before creating a new record, search by phone or email. Duplicate records are a data quality problem — always check for an existing lead first.
  2. If no record exists, create one. Fill all required fields immediately — don't create an empty skeleton and come back later.
  3. If a record exists, update it. Don't create a duplicate — update the existing one with your new information.

Required Fields (Minimum)

  • Name — First and last, correctly spelled and cased
  • Primary Phone — The number they answered or are likely to answer
  • Email — If available; don't leave blank if you have it
  • Lead Source — Where did this lead come from? (list, referral, web, inbound)
  • Status / Stage — Accurately reflects where they are in the pipeline
  • Assigned Owner — Who is responsible for this lead right now

B) Log a Call Attempt

  1. Add an activity of type Call to the lead record immediately after the attempt — not at end of day.
  2. Set the outcome code accurately:
    No Answer Callback Requested Qualified Not a Fit Do Not Contact
  3. Write your notes using this minimum standard:
    • What they want (goal)
    • Their timeline
    • Any objections raised
    • The agreed next step — what AND when
Note Quality

Avoid vague notes like "good call" or "seems interested." These tell the next person nothing. Your notes should let a teammate walk into the next call fully prepared without asking you anything.

C) Create a Follow-Up Task

  1. Task name: Action + context — e.g., "Call back — requested by lead" or "Send ARV analysis — promised on 4/15"
  2. Due date/time: Specific. Not "this week" — a real date and time slot.
  3. Reminder: Set per Ocean Luxe policy so it surfaces before it's due.
  4. Link to lead record: Always attach the task to the lead so anyone can find the context.

Data Quality Standards

  • Always include timeline and next step in notes when you have them
  • Use consistent casing and format for phone numbers and emails
  • Never use vague notes — be specific about what was said and agreed
  • Log even failed attempts (no answer, voicemail) — the timestamp matters
